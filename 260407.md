## Theorie
Man kann mit sachen wie `$0` oder `$1` auf verschiedene Argumente zugreifen die man selber übergibt. z.B.: dein Script heißt `myscript.sh` wenn du dann im Script `$0` verwendest, greift es auf das erste Argument zu das heißt `$0` = `myscript.sh`. Wenn du jetzt aber schreibst `myscript.sh abc xyz` dann wäre `abc` = `$1` und `$2` = `xyz`. Diese Variablen kannst du dann in Shell scripts verwenden Beispiel:

**TEST SHELL SKRIPT:**

```sh

# NAME = abc
NAME=$1
# NAME2 = xyz
NAME2=$2

# Erstellt verzeichniss mit Namen xyz
mkdir -p "$NAME2"

```

usw.


## 1. Übung Directory Struktur

**Erstell Skript**

```sh

#!/bin/bash

# schauen ob ein Argument übergeben wurde
if [ -z "$1" ]; then
    echo "Nutzung: $0 <name>"
    exit 1
fi

NAME=$1

# Das erstellt den Hauptfolder
mkdir -p "$NAME"

# Erstellt die zwei unterverzeichnisse im Hauptverzeichniss
for i in 01 02; do
    SUBDIR="${NAME}/${NAME}_${i}"
    mkdir -p "$SUBDIR"
    
    # Erstelle die 3 Textdateien pro Unterverzeichnis
    for j in 1 2 3; do
        touch "${SUBDIR}/${NAME}.${i}.${j}.txt"
    done
done

# Struktur anzeigen
echo "Struktur für '$NAME' wurde erstellt:"
tree "$NAME"

```
**Ausgabe:**

```sh

./build_dirs.sh abcd  
Struktur für 'abcd' wurde erstellt:
abcd
├── abcd_01
│   ├── abcd.01.1.txt
│   ├── abcd.01.2.txt
│   └── abcd.01.3.txt
└── abcd_02
    ├── abcd.02.1.txt
    ├── abcd.02.2.txt
    └── abcd.02.3.txt
```

**Lösch Skript:**

```sh

#!/bin/bash

# Schauen ob Argument übergeben wurde
if [ -z "$1" ]; then
    echo "Nutzung: $0 <name>"
    exit 1
fi

NAME=$1

# Das Löscht das Verzeichnis
if [ -d "$NAME" ]; then
    rm -rf "$NAME"
    echo "Verzeichnis '$NAME' wurde gelöscht."
else
    echo "Fehler: Verzeichnis '$NAME' existiert nicht."
fi

```

## 2. Übung Skript Generator

**Script:**

```sh

#!/bin/bash

echo "Wie heißt du?"

read NAME

echo "Hallo $NAME"
echo "Schön, dass du da bist."
```

mit `read` kann man abfragen und dann eingeben, dann wird die Variable befüllt

## 3. Übung Zeilenumbruch

**Script:**

```sh
MSG=" \n Name: Denis \n Klasse: 3AHITS \n Raum: ..."

echo $MSG
```

**Ausgabe:**
```sh
┌──(kali㉿kali)-[~/260324]
└─$ ./shell.sh

 Name: Denis 
 Klasse: 3AHITS 
 Raum: ...
```

Wie beim programmieren kann man mit `\n` ein zeilenumbruch machen

## 4. Übung admin

```sh

USER=admin

echo ${USER}_backup
echo ${USER}_2025

```
man braucht die Curly Braces `{}` weil die Shell sonst vermutet das du eine Variable `USER_backup` ausgeben willst.

## 5. Übung log files

```sh
DIR="/var/log/"

echo "${DIR}nginx"
echo "${DIR}apache"
```
Eigentlich genau das selbe wie bei Übung 4 nur das mit die Double Quotes nutzen muss

## 6 Übung backup

```sh
BASE="backup"
DATE="2026-01-16"

echo "${BASE}_${DATE}.tar"
echo "${BASE}_${DATE}.tar.gz"
```

Genau wie bei Übung 5 muss man darauf achten das man die Double Quotes und Curly Braces benutz
