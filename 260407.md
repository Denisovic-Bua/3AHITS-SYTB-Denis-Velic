## Theorie
Man kann mit sachen wie `$0` oder `$1` auf verschiedene Argumente zugreifen die man selber übergibt. z.B.: dein Script heißt `myscript.sh` wenn du dann im Script `$0` verwendest, greift es auf das erste Argument zu das heißt `$0` = `myscript.sh`. Wenn du jetzt aber schreibst `myscript.sh abc xyz` dann wäre `abc` = `$1` und `$2` = `xyz`. Diese Variablen kannst du dann in Shell scripts verwenden Beispiel:

**TEST SHELL SKRIPT:**

```sh

# NAME = abc
NAME=$1
# NAME2 = xyz
NAME2=$2

# Erstellt verzeichniss mit Namen xyz
mkdir -p "$NAME2"

```

usw.


## 1. Übung Directory Struktur

**Erstell Skript**

```sh

#!/bin/bash

# schauen ob ein Argument übergeben wurde
if [ -z "$1" ]; then
    echo "Nutzung: $0 <name>"
    exit 1
fi

NAME=$1

# Das erstellt den Hauptfolder
mkdir -p "$NAME"

# Erstellt die zwei unterverzeichnisse im Hauptverzeichniss
for i in 01 02; do
    SUBDIR="${NAME}/${NAME}_${i}"
    mkdir -p "$SUBDIR"
    
    # Erstelle die 3 Textdateien pro Unterverzeichnis
    for j in 1 2 3; do
        touch "${SUBDIR}/${NAME}.${i}.${j}.txt"
    done
done

# Struktur anzeigen
echo "Struktur für '$NAME' wurde erstellt:"
tree "$NAME"

```
**Ausgabe:**

```sh

./build_dirs.sh abcd  
Struktur für 'abcd' wurde erstellt:
abcd
├── abcd_01
│   ├── abcd.01.1.txt
│   ├── abcd.01.2.txt
│   └── abcd.01.3.txt
└── abcd_02
    ├── abcd.02.1.txt
    ├── abcd.02.2.txt
    └── abcd.02.3.txt
```

**Lösch Skript:**

```sh

#!/bin/bash

# Schauen ob Argument übergeben wurde
if [ -z "$1" ]; then
    echo "Nutzung: $0 <name>"
    exit 1
fi

NAME=$1

# Das Löscht das Verzeichnis
if [ -d "$NAME" ]; then
    rm -rf "$NAME"
    echo "Verzeichnis '$NAME' wurde gelöscht."
else
    echo "Fehler: Verzeichnis '$NAME' existiert nicht."
fi

```

## 2. Übung Skript Generator

**Generator:**

```sh
#!/bin/bash

# Schauen ob ein Argument übergeben wurde
if [ -z "$1" ]; then
    echo "Benutzung: $0 <skriptname>"
    exit 1
fi

NAME=$1
FILENAME="${NAME}.sh"

# Datei erstellen und inhalt hinzufügen
cat <<EOF > "$FILENAME"
#!/usr/bin/env sh
echo "$NAME Skript"
# write your script here
EOF

# Ausführungsrechte setzen
chmod +x "$FILENAME"

echo "Skript '$FILENAME' wurde erstellt und ist bereit."
```
EOF = End of File

Jetzt kannst du mit z.B.: `./generator.sh test` ein script erstellen

## 3. Übung Headline Cat

**Script:**

```sh
#!/bin/bash

# Schauen ob result.txt leer ist ( oder erstellt wird ) 
> result.txt

# Schleife über alle übergebenen Argumente mithilfe von $@
for file in "$@"
do
    # Trenner und Dateiname in die Zieldatei schreiben
    echo "== $file ==========================================" >> result.txt
    
    # Inhalt der aktuellen Datei an result.txt anhängen
    cat "$file" >> result.txt
done
```

**Ausgabe:**
```sh
┌──(kali㉿kali)-[~/260324]
└─$ ./shell.sh

 Name: Denis 
 Klasse: 3AHITS 
 Raum: ...
```

Wie beim programmieren kann man mit `\n` ein zeilenumbruch machen
