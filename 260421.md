## 1. ﾃ彙ung Anzahl Eintrﾃ､ge in einem Verzeichnis

Hier soll man ein shell script schreiben das als Argument einen Pfad auf ein Verzeichnis erhﾃ､lt. Das soll dann die Anzahl der Eintrﾃ､ge innerhalb diesem Verzeichnis ausgeben. Dabei sollte man fﾃｼr die Ausgabe `ls` und `wc` nutzen

**nbrentries.sh:**
```sh
#!/bin/bash

# erstes Argument in einer Variable speichern
DIR=$1

# Anzahl der Eintrﾃ､ge zﾃ､hlen
# ls listet auf wc -l zﾃ､hlt die Zeilen
COUNT=$(ls "$DIR" | wc -l)

echo "Es sind $COUNT Eintrﾃ､ge im dir $DIR"
```
**Ausgabe:**
```sh
./nbrentries.sh ‾/260324
Es sind 11 Eintrﾃ､ge im dir /home/kali/260324
```
Was ist der unterschied zwischen `ls`, `ls | cat` und `echo "$(ls)`?
- `ls`: Erkennt, dass es im Terminal ausgeben wird und sortiert dann die Namen in Spalten
- `ls | cat`: Schreibt jede Datei in eine eigene Teile untereinander, damit das nﾃ､chste Programm wie z.B.: `wc` sie einfach zﾃ､hlen kann
- `echo $(ls)`: nimmt die normale `ls` Anzeige als einen kompletten Textblock und gibt ihn genau so wieder aus

## 2. ﾃ彙ung Tage bis zum Ball

Hier muss man ein Skript schreiben was die Anzahl der Tage bis zum HTL Ball 2027 ermittel

**days_to_ball.sh:**
```sh
#!/bin/bash

#Zieldatum
BALL_DATUM="27-01-16"


#Umrechnungen vom Ball Datum in sekunden und jetzigen Datum
BALL_SEKUNDEN=$(date -d "$BALL_DATUM" +%s)
JETZT_SEKUNDEN=$(date +%s)

#Differenz berechnen
DIFFERENZ=$((BALL_SEKUNDEN - JETZT_SEKUNDEN ))

TAGE=$((DIFFERENZ / 86400))

echo "es sind noch "$TAGE" bis zum HTL-Ball am "$BALL_DATUM""
```
**Augabe:**
```sh
./days_to_ball.sh
es sind noch 276 bis zum HTL-Ball am 27-01-16
```
Das `date +%s` lieft dir die aktuelle Zeit in Sekunden, `date -d "27-01-16" +%s` wandelt ein spezifisches datum in Sekunden um


## 3. ﾃ彙ung Zuffﾃ､liger Satz
Erstelle ein Skript das einen Satz aus 5 zufﾃ､lligen Wﾃｶrtern bildet, z.B.:

`richtig ganzen Kilometer auf Mittel`


**Satz_generator:**

```sh
#!/bin/bash

# Datei mit der Wortliste
DATEI="wordlist.txt"

# 1. Anzahl der Wﾃｶrter dynamisch ermitteln (Erweiterung)
ANZAHL=$(wc -l < "$DATEI")

# Variable fﾃｼr den Satz
SATZ=""

# 2. 5 zufﾃ､llige Wﾃｶrter auswﾃ､hlen
for i in {1..5}
do
    # Zufallszahl zwischen 1 und ANZAHL
    INDEX=$(( ( RANDOM % ANZAHL ) + 1 ))
    
    # Wort mit sed extrahieren
    WORT=$(sed -n "${INDEX}p" "$DATEI")
    
    # Wort an den Satz hﾃ､ngen
    SATZ="$SATZ $WORT"
done

# 3. Ausgabe
echo "Satz:$SATZ"
echo "Anzahl der Wﾃｶrter in der Liste: $ANZAHL"
```

**Ausgabe:**
```sh
Satz: richtig ganzen Kilometer auf Mittel
Anzahl der Wﾃｶrter in der Liste: 622
```
