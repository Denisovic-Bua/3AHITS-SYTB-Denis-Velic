## Theorie:
### Developer Tool im Browser
Wenn man auf einem Browser die `F12` Taste drückt öffnen sicht ein Fenster unter Network sieht man dann den Trafic von HTTP-Requests vom Client und vom Server. Man kann auch manche Request genauer anschauen indem man drauf Klickt. Dannach öffnet sich eine Seite mit Kategorien wie `Headers`, `Cookies`,`Request`,`Response`

## 📁 cURL + Notenmanagment 

- Starte ein Browser z.B `Edge`, `FireFox` oder `Chrome`
- Öffne mit der `F12` Taste die Developer Tools
- ÖFfne die [Notenmanagment](https://notenmanagement.htl-braunau.at/) Seite
- Logge dich ein und öffne z.B.: ein Test der Angezeigt wird
-  Im Netzwerk tab suche nach dem API Request die als `JSON` übertragen werden
-  Dort Siehst du dann die angezeigen Daten die das Notenmanagement abruft
