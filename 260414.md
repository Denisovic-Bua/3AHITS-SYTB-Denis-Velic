## Theorie
Command Subtitution ist kurz gesagt einfach das man Kommand outputs in einer Variable speichern kann und z.B.: auch rechnungen damit machen kann



## 1. Übung Anzahl Einträge in einem Verzeichnis

Hier soll man ein shell script schreiben das als Argument einen Pfad auf ein Verzeichnis erhält. Das soll dann die Anzahl der Einträge innerhalb diesem Verzeichnis ausgeben. Dabei sollte man für die Ausgabe `ls` und `wc` nutzen

**nbrentries.sh:**
```sh
#!/bin/bash

# erstes Argument in einer Variable speichern
DIR=$1

# Anzahl der Einträge zählen
# ls listet auf wc -l zählt die Zeilen
COUNT=$(ls "$DIR" | wc -l)

echo "Es sind $COUNT Einträge im dir $DIR"
```
**Ausgabe:**
```sh
./nbrentries.sh ~/260324
Es sind 11 Einträge im dir /home/kali/260324
```
Was ist der unterschied zwischen `ls`, `ls | cat` und `echo "$(ls)`?
- `ls`: Erkennt, dass es im Terminal ausgeben wird und sortiert dann die Namen in Spalten
- `ls | cat`: Schreibt jede Datei in eine eigene Teile untereinander, damit das nächste Programm wie z.B.: `wc` sie einfach zählen kann
- `echo $(ls)`: nimmt die normale `ls` Anzeige als einen kompletten Textblock und gibt ihn genau so wieder aus

## 2. Übung Skript Generator

**Generator:**

```sh
#!/bin/bash

# Schauen ob ein Argument übergeben wurde
if [ -z "$1" ]; then
    echo "Benutzung: $0 <skriptname>"
    exit 1
fi

NAME=$1
FILENAME="${NAME}.sh"

# Datei erstellen und inhalt hinzufügen
cat <<EOF > "$FILENAME"
#!/usr/bin/env sh
echo "$NAME Skript"
# write your script here
EOF

# Ausführungsrechte setzen
chmod +x "$FILENAME"

echo "Skript '$FILENAME' wurde erstellt und ist bereit."
```
EOF = End of File

Jetzt kannst du mit z.B.: `./generator.sh test` ein script erstellen

## 3. Übung Headline Cat

**Script:**

```sh
#!/bin/bash

# Schauen ob result.txt leer ist ( oder erstellt wird ) 
> result.txt

# Schleife über alle übergebenen Argumente mithilfe von $@
for file in "$@"
do
    # Trenner und Dateiname in die Zieldatei schreiben
    echo "== $file ==========================================" >> result.txt
    
    # Inhalt der aktuellen Datei an result.txt anhängen
    cat "$file" >> result.txt
done
```



Wie beim programmieren kann man mit `\n` ein zeilenumbruch machen
